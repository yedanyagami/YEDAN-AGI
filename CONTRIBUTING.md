# Contributing to YEDAN AGI

First off, thank you for considering contributing to YEDAN.

## The Mission

YEDAN is not just a tool; it's a mission to democratize Autonomous E-commerce Intelligence. We believe that financial freedom should be accessible via code.

## How Can I Contribute?

### 1. Code Soldiers (Developers)
*   **Modules**: Improve our Python modules in `modules/`.
*   **Safety**: Enhance `modules/safety_guard.py` to prevent hallucinations.
*   **Connectors**: Add new API bridges (e.g., Stripe, Woo, Amazon).

### 2. Strategy Officers (Researchers)
*   Refine `strategies/` prompts for DeepSeek R1.
*   Improve market analysis algorithms.

### 3. Bug Hunters
*   Test the `run_roi_loop.py` cycle.
*   Report bugs via Issues.

## Pull Request Process

1.  Fork the repo and create your branch from `main`.
2.  If you've added code that should be tested, add tests.
3.  Ensure the test suite passes.
4.  Make sure your code lints.
5.  Issue that PR!

## Style Guide

*   **Python**: Follow PEP 8.
*   **Commits**: Use conventional commits (e.g., `feat: add new fast brain module`).
